¿Qué es programar?

La programación es el proceso de crear un conjunto de instrucciones que le dicen a una computadora como realizar algún tipo de tarea. Pero no solo la acción de escribir un código para que la computadora o el software lo ejecute. Incluye, además, todas las tareas necesarias para que el código funcione correctamente y cumpla el objetivo para el cual se escribió.

Para jugar debemos tener en cuenta las 3 secciones incluida esta. Vamos a presentar la información y después proseguiremos a presentarles un tablon de preguntas en una aplicación creada en estas aplicaciones.

Reglas:

  • Deben alzar la mano para participar en el test
  • Cuando les hagamos pasar deberán responder la pregunta, en caso de hacerlo mal haremos de nuevo un sorte en una aplicación de python
  • En caso de haber respondido correctamente se deberá se les dará dos opciones al participante:
    • A. Pasar al siguiente participante
    • B. Seguir jugando duplicando el premio
  • El sorteo se hará mediante un programa creado en python el cual nos tira numeros al azar. El participante que adivine el número tendrá la oportunidad de jugar
