Visual Basic

¿Qué es?

Es un software que nos permite crear aplicaciones de todo tipo, desde aplicaciones de cálculo hasta aplicaciones de teléfono. Este software fue diseñado en el lenguaje C++. Este lenguaje nos permite crear un interfaz y código el cual podremos editar a nuestra necesidad. Es especializado para la programación Orientada a objetos.

Fue creada por Alan Cooper y su primera versión se presenta en 1991Aunque Visual Basic es de propósito general, también provee facilidades para el desarrollo de aplicaciones de bases de datos usando Data Access Objects, Remote Data Objects o ActiveX Data Objects.

... La creación en este software es grandioso pero límitado, ya que sus componentes son bastantes antiguos, tomando en cuenta que la versión 6.0 es del 2007. Existen otras vesiones como Visual Basic.NEt, que de igual manera presenta algunos errores en la descarga o en la programación. El más optimo es el Visual Studio 2022 el cual es el software más completo de los ya nombrados, ofrece diferentes opciones de programación. Tiene ventajas únicas, lo que cambia es que este software es privativo lo que hace que tenga un precio, pero aún así nos ofecen una versión para la comunidad "VISUAL STUDIO 2022 COMUNITY" que se usa por un tiempo límitado de prueba y después un lapso de tiempo ya no podremos usarlo a menos de que paguemos la demanda de dinero que la empresa de Microsoft propone.

A lo largo del año electivo trabajamos en VISUAL basic, creamos varios programos: matemáticos, de lenguaje y de entreteniemiento. Ahora bien, para la creación de un programa en este software ya requiere de conocimientos previos. Se empiza desde lo más fácil: Psint y por consiguiente C++ y después VISUAL BASIC. Entonces te animamos a que sigas adelante con lo que te gusta y si es el caso de la informática te animamos a que tomes tú la iniciativa y te prepares en casa, empezando desde lo más sencillo, scratch, Pseint y los diagramas de flujo, C++, etc.
